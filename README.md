# ERC 20 Token

A Erc 20 token with transfer function between users.

## Run locally

Clone the repo

```
git clone https://github.com/shubhamkr95/ERC20_token.git
```

Go to project directory

```
cd ERC20_git
```

Install dependencies and dev-dependencies

```
npm install && npm install -D
```

Run the test file

```
npx hardhat test
```
